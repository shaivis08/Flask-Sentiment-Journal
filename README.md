# Flask Sentiment Journal

A simple web application built with **Flask** that allows users to write daily journal entries and track their mood. Sentiment analysis and mood visualization are planned for future updates.

## Features

- Add, edit, and delete journal entries.
- Clean and user-friendly interface using HTML and CSS.
- Mood tracking planned for future updates.

## Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS
- **Database:** SQLite
- **Libraries:** TextBlob (for sentiment analysis)
